# Muesli

A serial monitor with flavour.

Usage example

muesli --port COM5 --speed 57600 -f d

-p -port: com port to connect to. Default: COM4   
-b -baud: baudrate (default 57600)    
-f -format: set printing format h: hex d: decimal, a: ascii
